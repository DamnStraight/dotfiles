## Useful Web developement resources

Sites that make web design/develop process that much easier.

- [uiGradients](https://uigradients.com) - A collection of great looking CSS gradients.
- [PALX](https://palx.jxnblk.com) - Provide a color and get back a matching palette!
- [SVGBackgrounds](https://www.svgbackgrounds.com/) - Collection of customizable SVG backgrounds.
- [Photopea](https://www.photopea.com/) - Online photo editor. Has a great UI for those familiar with photoshop. Great for quick edits.
- [Icomoon](https://icomoon.io) - Let's you pick only the icons you need and generates a font for you, great when you don't want everything. (Note: A lot of icons are locked behind a paywall, but Font-Awesome icons are there!)
- [Feathericons](https://feathericons.com/) - Free icons that aren't Font-Awesome
- [Pexels](https://www.pexels.com/) - Free stock photos
- [Unsplash](https://unsplash.com/) - Free stock photos
- [unDraw](https://undraw.co/) - Free collection of great looking vector icons
